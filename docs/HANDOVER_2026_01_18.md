## 1. Bug Fix: FTS5 Empty Query

### Vấn Đề
```
sqlite3.OperationalError: fts5: syntax error near ""
```
Xảy ra khi Translation Memory nhận text không có keywords (empty string, special chars only).

### Giải Pháp
**File:** `core/translation_memory.py:340-354`

```python
# Before (bug)
keywords = self._extract_keywords(source)
fts_query = " OR ".join(keywords[:5])  # Empty if no keywords!
params.append(fts_query)

# After (fixed)
keywords = self._extract_keywords(source)
if keywords:
    query += " AND id IN (SELECT rowid FROM segments_fts WHERE segments_fts MATCH ?)"
    fts_query = " OR ".join(keywords[:5])
    params.append(fts_query)
else:
    return []  # Can't do meaningful fuzzy match without keywords
```

### Test
```python
python -c "
from core.translation_memory import TranslationMemory
tm = TranslationMemory(':memory:')
tm.get_fuzzy_matches('', 'en', 'vi')  # No crash!
tm.get_fuzzy_matches('!@#$%', 'en', 'vi')  # No crash!
"
```

### JavaScript
```javascript
// Theme toggle (Settings panel)
const STORAGE_KEY_THEME = 'ai-publisher-theme';

function applyTheme(theme) {
  if (theme === 'system') {
    document.documentElement.removeAttribute('data-theme');
  } else {
    document.documentElement.setAttribute('data-theme', theme);
  }
}
```
