## Three Core Pipelines

### 1. Translation Pipeline (Mature — v2.8+)
- **Smart Extraction**: PyMuPDF (text-only) → Vision LLM (scanned/formulas)
- **Parallel Translation**: `concurrency=5` via `core_v2/orchestrator.py`
- **Auto-Fallback**: OpenAI → Anthropic → DeepSeek
- **Performance**: 598 pages in ~28 min (was 4.5 hours), cost ~$0.28 (was $15-30)
- **Features**: Translation Memory, Glossary, Chunk Cache, Checkpointing

### 2. Book Writer v2 (Recent — Sprint 15+)
- **9-Agent Pipeline**: Analyst → Architect → Outliner → Writer → Expander → Enricher → Editor → QualityGate → Publisher
- **Parallelized**: 6 agents use `asyncio.Semaphore(5)` + `asyncio.gather()`
- **Features**: Draft upload, in-app reader, progress tracking
- **Known issue**: Older books have corrupted content ("Response object at 0x...") — fixed in `ai_adapter.py`, new books should work

### 3. Screenplay Studio (Newest — v3.3.0)
- **12 Agents**: StoryAnalyst, SceneArchitect, DialogueWriter, ActionWriter, Cinematographer, VisualDesigner, Storyboarder, VietnameseAdapter, PromptEngineer, VideoRenderer, VideoEditor, ScreenplayFormatter
- **Formats**: Fountain export, PDF export
- **Providers**: DALL-E, Runway, Pika, Veo (for visual generation)
- **Backend**: `api/routes/screenplay.py` (954 lines), `core/screenplay_studio/` (full module)
- **Frontend**: Dashboard, Editor, Storyboard, Video, Export, Wizard components
- **Latest**: PDF upload support added (commit `6da61cc`)
