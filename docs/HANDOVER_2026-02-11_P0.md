## Production Deployment Checklist
To deploy in production mode, set these env vars:
```bash
SECURITY_MODE=production
SESSION_SECRET=$(python -c "import secrets; print(secrets.token_hex(32))")
SESSION_AUTH_ENABLED=true
SESSION_BACKEND=file
CORS_ORIGINS=https://yourdomain.com,https://app.yourdomain.com
CSRF_ENABLED=true
CSRF_SECRET_KEY=$(python -c "import secrets; print(secrets.token_hex(32))")
```

The app will **refuse to start** if any of these are missing/insecure in production mode.
