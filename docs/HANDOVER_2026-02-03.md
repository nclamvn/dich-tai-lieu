## Thay Đổi Chính

### 1. Vision Provider Priority

**File:** `ai_providers/unified_client.py`

```python
# Text requests (thông thường)
PROVIDER_ORDER = ["openai", "anthropic", "deepseek"]

# Vision requests (STEM documents)
VISION_PROVIDER_ORDER = ["anthropic", "openai"]  # Claude first!
```

### 2. Logic Flow

```
📷 Vision Request (formula images):
   1️⃣ Claude Vision (claude-sonnet-4-20250514) ← Priority
   2️⃣ OpenAI Vision (gpt-4o) ← Fallback
   ❌ DeepSeek (no vision support)

📝 Text Request:
   1️⃣ OpenAI (gpt-4o-mini)
   2️⃣ Anthropic (claude-sonnet)
   3️⃣ DeepSeek (cheapest)
```

### 3. Auto-Fallback Triggers

| Error Type | Action |
|------------|--------|
| `credit balance is too low` | Switch provider |
| `invalid api key` | Switch provider |
| `rate_limit` | Retry same provider |
| Generic error | Switch provider |

## STEM Document Support

### Có Thể Xử Lý

| Loại | Không Vision | Có Vision |
|------|-------------|-----------|
| Text + LaTeX formulas | ✅ PyMuPDF | ✅ Perfect |
| Formulas dạng ảnh | ❌ Mất | ✅ Extract |
| Academic papers (arXiv) | ⚠️ Text only | ✅ Full |
| Scanned docs | ✅ OCR (free) | ✅ Vision |

### Provider Comparison

| Provider | Vision | Cost |
|----------|--------|------|
| Claude (Anthropic) | ✅ Yes | ~$0.02/page |
| OpenAI (GPT-4o) | ✅ Yes | ~$0.02/page |
| DeepSeek | ❌ No | $0.002/page |
