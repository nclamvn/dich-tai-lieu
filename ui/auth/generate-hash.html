PASSWORD HASH

Tạo SHA-256 hash cho mật khẩu

Bảo mật: Chỉ sử dụng tool này trên máy local. Không chia sẻ hash hoặc password với người khác.
SHA-256 HASH

Hướng dẫn sử dụng:

  1. Nhập mật khẩu bạn muốn sử dụng
  2. Click "TẠO HASH" để tạo SHA-256 hash
  3. Copy hash và paste vào file auth.js
  4. Tìm dòng PASSWORD_HASH: 'REPLACE_WITH_YOUR_HASH'
  5. Thay thế bằng hash vừa tạo
  6. Xóa file này sau khi hoàn tất
